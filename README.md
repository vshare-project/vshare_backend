# 🛵 RideX Backend API

Hệ thống Backend cho dự án RideX - Nền tảng cho thuê xe điện (xe đạp, xe máy) dành cho sinh viên tại các trường Đại học.

## 🛠 Công nghệ sử dụng
- **Core:** Node.js, Express, TypeScript
- **Database:** MySQL, TypeORM
- **Auth:** JWT (Access & Refresh Token), Bcrypt
- **Validation:** Class-validator
- **Docs:** Swagger UI

## 🚀 Cài đặt và Chạy dự án

### 1. Yêu cầu
- Node.js (v18 trở lên)
- MySQL Database
